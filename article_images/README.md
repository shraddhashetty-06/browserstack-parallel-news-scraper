# BrowserStack Parallel Execution with Selenium – News Scraper

This project demonstrates parallel test execution using BrowserStack Automate with Selenium and Python. It launches multiple browser sessions simultaneously and performs automated scraping of news articles from the El País website.

## Features

- Parallel execution on multiple browsers using BrowserStack
- Cross-browser testing (Chrome, Firefox, Edge, Safari)
- Automated scraping of news articles
- Spanish to English title translation using Deep Translator
- Automatic image extraction and download
- Word frequency analysis
- Local execution support
- Cloud execution support via BrowserStack

## Technologies Used

- Python 3.9
- Selenium WebDriver
- BrowserStack Automate
- Requests library
- Deep Translator
- Threading (parallel execution)
